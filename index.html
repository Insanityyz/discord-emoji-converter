How to Get Emoji IDs on Discord (Without or With Developer Mode)

On Both Mobile and PC:

  1. Ensure you are in the server that has the emoji you want to get the ID for.
  2. In the chat box, type \:youremojiname:, replacing youremojiname with the name of the emoji.
  3. Send the message. The ID of the emoji will appear in the chat as \.
  4. Copy the ID number from the message.
  5. Now using the converter, if the emoji is animated, use the GIF format in the dropdown, if not, use WEBP.
  6. Once done, insert your copied emoji id into its respective placebox. Then click Convert!
  7. Once converted, just double click the link that has been created and copy it, once done, you can post it anywhere, and it'll turn into your EMOJI!

Note: This method only works if you are in the server that has the emoji. If you are not in the server, you won't be able to retrieve the emoji ID using this method.

How to Enable Developer Mode on Discord (if you're unsure or if it doesn't work somehow)

On Mobile:

  1. Open the Discord app on your mobile device.
  2. Tap on your profile picture in the bottom right corner to open User Settings.
  3. Scroll down and select Advanced.
  4. Toggle the Developer Mode switch to enable it.

On PC:

  1. Open Discord on your PC and log in.
  2. Click on the gear icon (User Settings) at the bottom left corner next to your username.
  3. Scroll down and select Advanced in the left sidebar.
  4. Toggle the Developer Mode switch to enable it.
